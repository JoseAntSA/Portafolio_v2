Sobre Mí

Hola, me llamo Antonio. Soy recién egresado en ingeniería en computación, actualmente me encuentro en proceso de titulación. Soy una persona curiosa y apasionada por su carrera que le encanta aprender nuevas cosas, además me gusta implicarme mucho en mi trabajo para poder realizar bien las cosas. Mi formación como ingeniero fue en el Centro Universitario UAEM Zumpango en donde adquirí habilidades como programador en distintos lenguajes, así como también la documentación de código.

Mi servicio social lo realice como parte de una brigada multidisciplinaria que se enfocó en la investigación y divulgación informativa a través de redes sociales, sobre temas de programación y matemáticas con ayuda de videos e infografías.

Actualmente formo parte del programa ONE (Oracle Next Education) iniciando una nueva etapa del programa en donde realizare mi especialización como desarrollador Front-End.

Tecnologías

Las siguientes son algunas de las tecnologías con las que mas he trabajado y lo que he hecho con ellas.

Java

He trabajado con hilos, sockets, interfaces gráficas con la clase swing. Los trabajos que he hecho con este lenguaje han sido la implementación de algoritmos genéticos, algunos algoritmos de clasificación como KNN y K-Means, así como también desarrollar un analizador léxico sintáctico capaz de leer código en lenguaje Python.

Python

Con Python trabaje el tratamiento en imágenes digitales desarrollando los métodos para poder comprender mejor como funcionaban técnicas de procesamiento digital. También vi con este lenguaje principios de Graficación como por ejemplo, curvas de Bézier y transformaciones de figuras. También desarrollé algoritmos genéticos e hice uso de redes neuronales para la implementación de un traductor de lenguaje de señas en tiempo real.

MySQL

La gestión de bases de datos la llevé a cabo con SQL con ayuda del gestor MySQL, con este lenguaje aprendí a manipular las tablas de las bases de datos, así como también, desarrollar funciones vistas y procedimientos para las propias bases de datos.

Linux

Siendo Debian, Kali y principalmente Ubuntu las distribuciones con las que trabaje, dentro de ellas programe scripts con Bash, realice la creación y gestión de usuarios, así como también de claves públicas y privadas, realizando de la mano igual con la gestión de permisos.

GitHub

Para facilitar la organización de proyectos y permitir la colaboración en tiempo real cuando trabajaba en equipo use GitHub para crear repositorios de código y guardarlos en la nube de forma segura, usando el sistema de control de versiones.

Arduino

Con Arduino desarrolle un robot mini sumo como proyecto final de robótica, al igual que también fabrique un carro a control remoto que era conducido desde del celular cuya interfaz fue desarrollada con el software de App Inventor.

Trello

Trello lo use para poder gestionar mis proyectos en equipo y flujos de trabajo, así como supervisar las tareas.

Jira

También llegue a trabajar con Jira para la gestión de trabajo en equipo y así poder organizar y hacer un seguimiento de nuestro trabajo.

HTML/CSS

He desarrollado páginas web sencillas para algunos de mis proyectos que involucraban una base de datos, tal es el caso de una página de login que conecte con una base de datos y le agregue el manejo de sesiones con PHP. Aun sigo en aprendizaje para poder trabajar de mejor manera con este lenguaje de marcado.

Figma

Durante mi curso de desarrollo móvil en Android, aprendí las bases de diseño de interfaz para poder realizar moodboards de proyectos, así como los wireframes. También tengo noción sobre los diferentes perfiles de UI y como participan con el UX.

MPLAB IDE

Con MPLAB trabaje la programación de los PIC 16F84 y 16f887 diseñando algunos programas simples, como lo es encender una serie de LEDS formando recorridos, diseñar un contador con displays o el diseño de un letrero movil en una pantalla LCD.
